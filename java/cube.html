Here is a 3D cube applet written in Java. I used the Java graphics primitives drawPolygon() and fillPolygon() to draw the cube. All transformations and renderings are generated dynamically using a matrix class and point class I wrote. The hidden line removal technique is done by sorting the sides by Z coordinate and drawing the visible sides over the top of the ones below.

The initial view will be a flat square. Press the Rotate buttons to see it as a 3D cube.  The controls on the right can change the position, size, and rotation on the X, Y, and Z axes. This is an applet I wrote for an RIT class, Topics in Graphics in August 1996.
