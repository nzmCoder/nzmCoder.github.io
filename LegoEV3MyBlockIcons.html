Lego EV3 Custom MyBlock Icons

To Make Custom MyBlock images for EV3

  1. The image icon files for MyBlocks are stored at:  C:\Program Files (x86)\LEGO Software\LEGO MINDSTORMS EV3 Home Edition\Resources\MyBlocks\images
  2. If your computer does not show a preview of *.PNG files in Windows Explorer, fix that problem first:
    • In Explorer, select a PNG file
    • Go to Tools menu (ALT key shows menu), Folder Options, View tab.
    • The option "Always Show Icons, Never Previews", untick that item.
  3. There are 3 files for each icon, a Diagram, a Palette, and a Palette_MouseOver.
  4. Copy and paste a set of those files, any existing set of 3, as a starting point.
  5. Rename the files to have new names.
  6. Edit the graphic images using any graphics editor, like Windows Paint.
  7. Save your work.
  8. Now, restart EV3 software, and you should be able to select the new icons.
